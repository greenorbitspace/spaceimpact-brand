# Leicester Space Week Brand Portal

The Astro source for `brand.leicesterspaceweek.com`.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The static deployment output is created in `dist/`.

## Structure

- `src/pages/` — individual brand-guideline routes
- `src/components/` — reusable editorial and navigation components
- `src/layouts/` — shared application shell
- `src/styles/global.css` — design tokens and responsive styling system
- `public/assets/logos/` — downloadable SVG logo assets

## Routes

- `/` — brand overview
- `/logo/` — primary logo, clear-space, sizing and misuse rules
- `/colours/` — approved palette and contrast guidance
- `/typography/` — type families and scale
- `/visual-language/` — motifs and photography direction
- `/applications/` — digital, social, physical and partner use
- `/downloads/` — asset download hub
- `/developers/` — CSS design tokens
